如何复制Windows命令行中的文本

发表于 2016-08-08 | 更新于 2018-06-24 | 分类于 Windows

有时候需要在复制Windows的命令行中的一些内容，这个时候我们平时使用的CTRL+C命令就不适合了，因为在命令行里面的作用是直接中断。

一个简单的复制方法就是首先用鼠标选中需要复制的文本，然后按Enter按键（也就是回车）即可。
